How I Built This Portfolio

Date: April 7, 2025

Creating my portfolio website was an exciting journey that allowed me to apply my knowledge of HTML and CSS while improving my design and structuring skills. Since I am still learning and do not have projects to showcase yet, I decided to document my progress through this blog. Here’s how I built my website and what I learned along the way..

Structuring the Website with HTML

Creating my portfolio website was an exciting journey that helped me apply my HTML and CSS knowledge. I divided the site into multiple sections, such as Home, About, Blog, and Contact, ensuring a clear structure using semantic elements like 'header' and 'section'.

Styling the Website with CSS

To make my website visually appealing, I focused on CSS styling. I implemented a consistent color scheme, used Flexbox for layout, and added hover effects to enhance interactivity. CSS transitions made elements appear smoothly, creating a more polished look.

Adding Personal Touches

Since I don't have projects to showcase yet, I decided to make my portfolio engaging by writing this blog, adding an image of myself, and including a disclaimer about my learning progress. This keeps the site meaningful while I work on future projects.

Lessons Learned and Future Goals

This project helped me improve my HTML & CSS skills and understand web design principles. Moving forward, I plan to build more projects, learn new technologies, and continuously update my portfolio to reflect my growth in web development.
